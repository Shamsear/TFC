## How to Use

### For Users (Public)
1. Navigate to `/news` from the main menu
2. Browse published news articles
3. Toggle language (EN ↔ ML)
4. Filter by category

### For Admins
1. Navigate to `/super-admin/news` from admin menu
2. Review AI-generated drafts
3. Publish or delete articles
4. Manage published content

### For Testing
1. Navigate to `/super-admin/test-news`
2. Click any event button to test AI generation
3. Review the generated bilingual content
4. Go to News Management to publish
