# Database Cleaning Tools

Tools to clean and prepare eFootball databases before importing to the main application.

## Why Clean the Database?

- **Remove duplicates** - Same player appearing multiple times
- **Fix invalid data** - Missing names, positions, or ratings
- **Normalize data** - Standardize positions and trim whitespace
- **Reduce size** - Smaller files upload faster
- **Improve quality** - Only valid, unique players
